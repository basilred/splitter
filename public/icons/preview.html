Использование в проекте

Иконка расположена в public/icons/split-bill-icon.svg. Вы можете использовать её как:

  • Иконка бота в Telegram
  • Favicon веб-интерфейса
  • Логотип в заголовке страницы
  • Графический элемент в UI

Пример вставки в React-компонент:

import SplitBillIcon from '/icons/split-bill-icon.svg';

function AppIcon() {
  return ;
}

Или как фондовое изображение в CSS:

.bot-icon {
  background: url('/icons/split-bill-icon.svg') center/contain no-repeat;
  width: 48px;
  height: 48px;
}

Иконка создана для проекта Telegram-бота разделения счёта. Формат: SVG, размеры: 128×128 пикселей.

Рекомендуется использовать с тёмным или светлым фоном для лучшей контрастности.
